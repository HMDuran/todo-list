# ✅ To-Do List

A simple To-Do List app using **JavaScript, HTML, and SASS**.

## 🚀 Features
- ➕ Add tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 📱 Responsive design

## 🛠 Technologies Used
- 🏗 **HTML** - Structure
- 🎨 **SASS** - Styling
- ⚡ **JavaScript** - Functionality
